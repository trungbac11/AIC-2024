### Setup 

```bash
python -m venv myenv 
.\myenv\Scripts\activate 
python.exe -m pip install --upgrade pip
pip install -r requirements.txt
```
### Run
```bash
.\myenv\Scripts\activate
cd app
uvicorn main:app --reload
```
